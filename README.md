# Bob's-Solicitors
While not built for a specific law firm, I wanted this website to look professional, but while adding in a bit of elegance and style.

## Sections
### Hero
The hero section on this website is incredibly minimal, but exceptionally impactful. The combination of the bold blue background and the full-screen design helps it to stand out amongst a sea of generic websites.

## Practice Areas
A simple section that shows the various practice areas of Bob's Solicitors, complete with hover effects. In a full version of this site, clicking these cards would take you to a page dedicated to that practice area.

## Office Locations
Making use of a tried-and-tested grid layout, this section includes all the information you'd need to know about each office, alongside a helpful, interactive, map.

## Contact & Footer
Finishing off the site, there is a small contact call-to-action, which allows the user to draft an email to the relevant contact email address. This is just above the footer which contains all relevant links, as well as social media profiles.
